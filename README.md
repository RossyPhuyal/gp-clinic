# GP Clinic Charges Dashboard

A dockerized app with Angular + AG-Grid, FastAPI, and MySql built in MVC framework.

## Quick Start

```bash
docker-compose up --build
```

## Backend runs in: http://localhost:8000
 - Health Check API: http://localhost:8000/health

## Frontend runs in: http://localhost:4800

## Database config: 
    - host: localhost
    - port: 3310
    - user: gpclinic_user
    - password: gpclinic_pass
